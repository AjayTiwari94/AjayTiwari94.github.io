# Ajay Tiwari - Portfolio

Personal portfolio site for Ajay Tiwari, Data & Business Analyst.

**Live site:** [ajay-tiwari94-portfolio.vercel.app](https://ajay-tiwari94-portfolio.vercel.app)

## About

B.Tech CSE (Artificial Intelligence) graduate showcasing data analytics projects, technical skills, professional experience, and education. Built as a static multi-page site with no framework dependencies.

## Pages

- **Home** - introduction, key stats, and skill highlights
- **Skills** - technical stack broken down by category
- **Experience** - internships and professional history
- **Projects** - data analytics projects with tools, findings, and impact
- **Education** - academic background and certifications
- **Contact** - get in touch

## Tech Stack

- HTML5, vanilla JavaScript
- [Tailwind CSS](https://tailwindcss.com/) (via CDN)
- [Web3Forms](https://web3forms.com/) for the contact form
- Deployed on [Vercel](https://vercel.com/)

## Running Locally

No build step required - it's static HTML/CSS/JS.

```bash
git clone https://github.com/AjayTiwari94/<repo-name>.git
cd <repo-name>
```

Then open `index.html` in a browser, or serve the folder with any static server:

```bash
npx serve .
```
